# web-crawler-and-scraper-rag
A rag chatbot with the ability to crawl and scrape links, process them and answer questions based on the documents
##Steps to run the app
1) Create a virtual environment
2) pip install langchain langchain-community langchain-core langchain-chroma langchain-google-genai streamlit python-dotenv beautifulsoup4 requests chromadb pypdf2 docx2txt pandas
3) streamlit run streamlit_app.py
